🎄 **Advent of Code 2025**

This repository contains my solutions to Advent of Code 2025, implemented in Python.

📌 **About Advent of Code**

Advent of Code is an annual Christmas-themed challenge held every December. A new problem is released each day, following the Advent calendar format. The problems cover a wide range of topics, including data structures and algorithms, recursion, memoization, linear programming optimization, etc. 

🐍 **About This Repository**

This repository documents my journey solving the 2025 edition of Advent of Code.
* All solutions are written in Python
* Some solutions are entirely my own
* Some are inspired by community discussions and online solutions
* Focus is on learning, improving problem-solving skills, and writing clean, readable code

📂 **Repository Structure**
* DayXX.py → Python implementation for that day's challenge (covers Part 1 and/or Part 2).
* input.txt → Input file used for solving the problems.
  - A single global input.txt is maintained.
  - The relevant input for each day is copied and updated accordingly before execution.
